# My New Website

This is the starting point for my personal website, created with Cursor.

## Getting started

- **Open the project**: Open the `c:\samocvetcom` folder in Cursor.
- **Run locally**:
  - Option 1 (recommended): Use the built-in `Open with Live Server` / `Simple Web Server`–style extension if you have one.
  - Option 2: On Windows, you can run a quick static server with Python:
    - `python -m http.server 8000`
    - Then visit `http://localhost:8000` in your browser.

## Files

- `index.html`: Main page structure.
- `styles.css`: Styles for the site.
- `script.js`: Small interactive behavior.

You can start by editing the text in `index.html` and the colors/layout in `styles.css`.
